# Shubh Desai – Portfolio Website

A personal portfolio website showcasing my experience, projects, and skills as a Computer Science undergraduate at the University of Michigan. This site is designed with a clean, modern UI and is fully responsive across devices.

---

## 🚀 Features
- Responsive design for desktop, tablet, and mobile devices
- Interactive navigation bar with smooth section scrolling
- Experience section highlighting professional background
- Projects section with detailed descriptions and technology icons
- Contact form integrated with Web3Forms
- Social media links and downloadable resume

---

## 🛠️ Tech Stack
- **HTML5** – Semantic structure and layout
- **CSS3** – Styling, layout, and responsiveness
- **Remix Icons** – UI icons
- **Devicon** – Technology icons
- **Web3Forms** – Contact form backend service
- **JavaScript** – Typing animation, DOM manipulation, UI behavior
